understanding HTML tables

In HTML,tables are created using the ⁢table>tag. the table is structured with rows(⁢tr>)and cells,which can be either header cells (⁢th>)or data cells(⁢td>). Attributes like border,rowspan,and colspancan be used to enhance the tables appearance and structure.

Therowspan attribute allows a cell to span multiple row.

Table with colspan

The colspan attribute allows a cell to span multiple columns
